ORACLE_BASE
Base directory for Oracle software and diagnostic files (varies by install).

ORACLE_HOME
Directory where Oracle binaries and libraries are installed.

ORACLE_SID
Instance identifier for the database on the server.

. oraenv
Shell script to set Oracle environment variables for a chosen SID.

/etc/oratab
File mapping SIDs to ORACLE_HOME paths; often used by oraenv/dbstart/dbshut.
